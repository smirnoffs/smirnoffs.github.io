Problem with six package on El Capitan

08 January 2016

I was trying to install Virtualenv Wrapper on my new Macbook and pip failed on installing six. Turns out that El Capitan has its own six installed in system protected folder.

The solution is next:

pip install virtualenvwrapper --ignore-installed six
